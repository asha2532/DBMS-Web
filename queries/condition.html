ITERATIVE CONTROL STRUCTURE

In PL/SQL, iterative control structures allow you to execute a block of code repeatedly until a certain condition is met. The primary construct for iterative control in PL/SQL is the LOOP statement. Additionally, PL/SQL also supports FOR LOOP, which is a specialized form of iterative control.

1.INFINITE LOOP

The LOOP statement repeatedly executes a block of code until an exit condition is met. The exit condition is typically determined using an EXIT statement within the loop.

syntax..

LOOP
-- Code block to execute repeatedly
EXIT WHEN condition; -- Exit condition
END LOOP;

example..

DECLARE
counter NUMBER := 1;
BEGIN
LOOP
DBMS_OUTPUT.PUT_LINE('Counter: ' || counter);
counter := counter + 1;
EXIT WHEN counter > 5; -- Exit loop when counter exceeds 5
END LOOP;
END;

2.FOR LOOP

The FOR LOOP statement is a specialized form of iterative control that iterates over a range of values. It automatically initializes a loop index variable, checks a condition, and increments the loop index variable with each iteration.

syntax..

FOR loop_index IN [REVERSE] lower_bound..upper_bound LOOP
-- Code block to execute for each iteration
END LOOP;

example..

DECLARE
-- Loop over the range of numbers from 1 to 5
BEGIN
FOR i IN 1..5 LOOP
DBMS_OUTPUT.PUT_LINE('Number: ' || i);
END LOOP;
END;

3.WHILE LOOP

PL/SQL while loop is used when a set of statements has to be executed as long as a condition is true, the While loop is used. The condition is decided at the beginning of each iteration and continues until the condition becomes false.

syntax...

WHILE
LOOP statements;
END LOOP;

example..

DECLARE
i INTEGER := 1;
BEGIN
WHILE i <= 10 LOOP
DBMS_OUTPUT.PUT_LINE(i);
i := i+1;
END LOOP;
END;
