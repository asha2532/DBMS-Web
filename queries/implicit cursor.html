Implicit Cursors

These cursors are automatically created by the database system for SQL statements that don't explicitly use cursor declarations. They are used for simple queries that return a single result set.

Syntax..

BEGIN
SQL statement; -- The SQL statement that returns a result set -- Processing logic here
END;

Example Program

student table
rollno name ph no. class marks city
4201 abhishek 1234567867 bca-1 296 chandigarh
4202 anjali 4567891234 bca-1 286 mohali
4203 asha 7719406253 bca-1 280 haryana
4204 neha 8876543219 bca-1 276 punjab

DECLARE
noOfRows number;
BEGIN
update student set class = 'BCA-II' where marks > 280;
noOfRows := SQL%ROWCOUNT;
dbms_output.put_line(noOfRows || ' Rows Updated Successfully');
END;

Output

3 Rows Updated Successfully
