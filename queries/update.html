SQL UPDATE Statement

The UPDATE statement is used to modify the existing records in a table.

query for update the table.....

update student set marks=92 where sname="Annie"; update student set marks=90where sname="danny"; update student set marks=89 where sname="alana"; update student set marks=75 where sname="robert"; update student set marks=80 where sname="maria";

all rows updated succesfully...
