MySQL WHERE Clause

The WHERE clause is used to filter records. It is used to extract only those records that fulfill a specified condition.

Query for WHERE clause...

select name from student where roll_no=7072;

Output

Sname roll_no age loc ph_no marks
Annie 7072 20 dallas 1234567897 92

select Sname from student where loc=dallas;

Output

Sname
Annie
robert
