SQL INNER JOIN

The INNER JOIN keyword selects records that have matching values in both tables.

table empDetails..
ID name salary
1 john 40000
2 alex 25000
3 simon 42000
table maritalStatus..
ID name status
1 john married
3 simon married/td>
4 stella unmarried

query to perform inner join....

select ID,name,salary, status, from table empDetails inner join table maritalStatus on table empDetails.ID=maritalStatus.ID

Output

ID name salary status
1 john 40000 married
3 simon 42000 unmarried
