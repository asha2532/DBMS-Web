SQL Views

Views in SQL are a kind of virtual table. A view also has rows and columns like tables, but a view doesnâ€™t store data on the disk like a table. View defines a customized query that retrieves data from one or more tables, and represents the data as if it was coming from a single source. We can create a view by selecting fields from one or more tables present in the database. A View can either have all the rows of a table or specific rows based on certain conditions.
