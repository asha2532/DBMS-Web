SQL FULL OUTER JOIN Keyword

The FULL OUTER JOIN keyword returns all records when there is a match in left (table1) or right (table2) table records. Tip: FULL OUTER JOIN and FULL JOIN are the same.

table one...
roll_no name
1 meenakshi
2 manjli
3 shetal
4 apporva
5 keeto
6 nitin
7 soumya
table two...
roll_no city
1 ajmer
3 banglore
5 udaipur
6 chitor
10 raipur

query to perform full join...

select name,city from table one full outer join table two on table one.name=table two.name order by city;

Output

name city
meenakshi ajmer
manjli NULL
sheetal banglore
apporva NULL
keeto udaipur
nitin chitor
soumya NULL
NULL raipur
